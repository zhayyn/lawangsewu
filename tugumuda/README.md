# TUGUMUDA

Aplikasi mini di folder `tugumuda` untuk mengambil data dari server 10.

## File

- `index.php` : dashboard
- `proxy.php` : backend proxy
- `config.php` : host, token, timeout, whitelist endpoint

## URL akses

`/tugumuda/index.php`

## Default endpoint

- `/lumpiapasar/api/monitor`

Jika endpoint berbeda, ubah di input halaman atau tambahkan whitelist di `config.php`.
