# Lawangsewu

Proyek web sederhana untuk menampilkan informasi antrian sidang.

Quick start

- Jalankan server web di folder ini (mis. Apache/Nginx).
- Pengelolaan versi ada di GitHub: https://github.com/zhayyn/lawangsewu

Files of interest

- `index.php` — entry point
- `antrian_controller.php` — controller generator slide
- `widgets/views/antrian-sidang.php` — view widget

Notes

- Jangan simpan kunci privat di repositori. Saya akan membuat kunci deploy lokal dan tunjukkan kunci publiknya untuk Anda tambahkan ke GitHub (Settings → Deploy keys).
